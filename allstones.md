| **重压** | DEBUFF，可叠加。每层降低目标移动速度和少量固定防御力。 |
| **石化** | 硬控。单位无法移动和攻击，但防御力大幅提升。 |
| **破甲** | DEBUFF，可叠加。按百分比降低目标防御力。 |

| **感电** | DEBUFF，可叠加。目标每次受到攻击时，会额外承受一次基于“感电”层数的雷电伤害。 |

| **致盲** | DEBUFF。目标攻击时有50%几率未命中。 |
